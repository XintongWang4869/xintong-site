# Personal Website

This is the source code for my personal website (**Live Site:** [https://xintong-wang.netlify.app/](https://xintong-wang.netlify.app/)). 

* **Static Site Generator:** [Hugo](https://gohugo.io/) 
* **Theme:** [Hugo Profile](https://github.com/gurusabarish/hugo-profile) 
* **Markup Languages:** Markdown & AsciiDoc 
* **Hosting & CI/CD:** Netlify


## Project Structure

Here is a brief overview of the directory structure relevant to content management:

```
.
├── content/           # Website content (Markdown/AsciiDoc files)
│   ├── blogs/         # Blog articles
│   └── portfolio/     # Technical documentation samples
├── layouts/           # HTML templates and overrides
├── static/            # Images and custom CSS files
└── hugo.yaml          # Main configuration (homepage params, menu, and other settings)

```
